## CONTESTO GENERALE

Sistema per mantenere cache di voci recenti di Wikipedia italiana:
- **Bot principale**: Scarica voci recenti da RecentChanges e le salva in moduli Lua
- **PuliziaCache.py**: Script separato per pulizia cache (duplicati, NS errati, voci cancellate)
- **Cache manuale**: Pagina wiki per voci spostate da sandbox non rilevate automaticamente

### C. IMPLEMENTARE logica CacheParsed

**Scopo:** Evitare di processare più volte la stessa cache manuale

**Logica richiesta:**
```python
def check_should_load_manual_cache():
    """
    Verifica se caricare la cache manuale
    Returns: True se deve essere caricata, False altrimenti
    """
    try:
        page = pywikibot.Page(SITE, CACHE_PARSED_PAGE)
        
        # Se non esiste → deve essere caricata
        if not page.exists():
            return True
        
        # Se esiste ma contenuto ≠ "True" → deve essere caricata
        content = page.text.strip()
        if content != "True":
            return True
        
        # Altrimenti già processata
        return False
    except:
        # In caso di errore, carica per sicurezza
        return True


def mark_manual_cache_as_parsed():
    """
    Marca la cache manuale come già processata
    """
    try:
        page = pywikibot.Page(SITE, CACHE_PARSED_PAGE)
        page.text = "True"
        page.save(summary="Bot: Cache manuale processata", minor=True)
        print(f"  ✓ Marcata cache manuale come processata")
    except Exception as e:
        print(f"  ERRORE marcatura: {e}")
```

**Integrazione in main():**
```python
# DOPO aver caricato la cache esistente:

should_load = check_should_load_manual_cache()

if should_load:
    print("Caricamento cache manuale...")
    moved_pages = read_moved_pages_list(existing_titles)
    
    if moved_pages:
        cached_pages.extend(moved_pages)
        # Aggiorna existing_titles per evitare duplicati
        for p in moved_pages:
            existing_titles.add(p['titolo'])
        
        # Marca come processata
        mark_manual_cache_as_parsed()
else:
    print("Cache manuale già processata - skip")
```

## FLUSSO LOGICO v8.0

```
1. Login
2. [Opzionale] RUN_CLEANUP=True → Esegui PuliziaCache.py
3. Carica cache esistente → cached_pages, existing_titles
4. Verifica CacheParsed:
   - Se non esiste O ≠ "True":
     a. Carica CacheMoved
     b. Aggiungi a cached_pages (solo se non in existing_titles)
     c. Aggiorna existing_titles
     d. Marca CacheParsed = "True"
   - Altrimenti: skip
5. Trova timestamp range (più vecchio/recente)
6. Cerca nuove voci da RecentChanges
   - Per ogni voce: verificare if title not in existing_titles
   - Dopo aggiunta: existing_titles.add(title)
7. Unisci: cached_pages + new_pages
8. Ordina per timestamp (più recenti prima)
9. Mantieni top MAX_PAGES
10. Salva in file multipli
```

## FORMATO CACHE MANUALE

**File:** `Utente:BotVociRecenti/CacheMoved`

**Formati supportati:**

1. **Titolo semplice:**
```
Grimpella thaumastocheir
Megaleledone setebos
```

2. **Dump da Speciale:PagineRecenti:**
```
16:32, 28 feb 2026 Grimpella thaumastocheir (cron | modifica) [3 657 byte]
11:25, 28 feb 2026 Megaleledone setebos (cron | modifica) [4 806 byte]
```

**Parsing:** Regex per estrarre titolo da dump:
```python
match = re.search(r'\d{2}:\d{2}, \d{1,2} \w+ \d{4} (.+?) \(cron \|', line)
if match:
    title = match.group(1).strip()
```

**Verifica namespace:** Se titolo estratto è in altro NS (es. Utente:), cercare in NS0:
```python
temp_page = pywikibot.Page(SITE, title)
if temp_page.namespace() != 0:
    base_name = temp_page.title(with_ns=False)
    if '/' in base_name:
        base_name = base_name.split('/')[-1]
    voce_page = pywikibot.Page(SITE, base_name, ns=0)
```

```python
def escape_for_lua_longstring(s):
    level = 0
    while f"]{'=' * level}]" in s:
        level += 1
    open_delim = f"[{'=' * level}["
    close_delim = f"]{'=' * level}]"
    return open_delim, close_delim
```
